プライバシーポリシー

最終更新日: 2026年7月19日

「オープンワールド将棋」(以下「本サービス」)は、個人が非商用で運営するブラウザゲームです。本ポリシーは、本サービスの利用にあたって取り扱われる情報の内容と範囲について説明するものです。

1. 運営者が意図的に収集する情報

本サービスは、アカウント登録・ログイン機能を持ちません。氏名、メールアドレス、電話番号、住所、決済情報などの個人情報を、運営者が意図的に収集することはありません。

ネット対戦モードをご利用の際、プレイヤーご自身が任意で入力する「表示名」(最大10文字、不適切な語句は自動フィルタ)のみを取得します。この表示名は対戦中の識別のためだけに使われ、対戦サーバーのメモリ上に一時的に保持され、永続的なデータベースには保存されません。対戦部屋が空になると、一定時間後に自動的に破棄されます。

2. Cookie・ローカルストレージについて

本サービスは、Cookie、localStorage、sessionStorageなどのブラウザ保存機能を使用していません。トラッキング目的の識別子も発行していません。

3. 通信・ホスティングに伴う技術的情報

本サービスの性質上、以下のような技術的な通信情報が、運営者以外の第三者インフラ事業者によって標準的なアクセスログとして記録される場合があります。これは本サービス固有の仕組みではなく、インターネット上の通信・ホスティングにおいて一般的に発生するものです。

サービス役割記録されうる情報
GitHub Pagesゲーム画面(静的ファイル)の配信アクセス元IPアドレス等の標準的なWebサーバーログ
Render.com対戦サーバー(WebSocket)の稼働接続元IPアドレス、接続時刻等の標準的なサーバーログ
jsDelivr(CDN)React等のライブラリ配信アクセス元IPアドレス等の標準的なCDNログ

これらの情報は各インフラ事業者が自社のサービス運営・不正防止のために保持するものであり、運営者が個人を特定する目的で取得・分析することはありません。各社のプライバシーポリシーは以下をご参照ください。

  • GitHub プライバシーステートメント
  • Render プライバシーポリシー
  • jsDelivr プライバシーポリシー

4. 広告・アクセス解析

本ポリシー公開時点で、本サービスは広告配信サービスおよびアクセス解析ツール(Google Analytics等)を導入していません。将来的に導入する場合は、本ポリシーを事前に改定し、その内容を明示します。

5. 未成年の利用について

本サービスは年齢確認を行っておらず、未成年の方もご利用いただけますが、氏名等の個人情報の入力を求める設計にはなっていません。保護者の方は、お子様がインターネット上のサービスを利用する際の一般的な注意を払っていただくことを推奨します。

6. 情報の第三者提供

運営者は、法令に基づく場合を除き、取得した情報を第三者に提供することはありません。

7. 本ポリシーの変更

本ポリシーの内容は、法令の変更やサービス内容の変更に応じて、予告なく改定される場合があります。改定後の内容は本ページに掲載した時点から効力を生じます。
